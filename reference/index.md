# Reference

Everything you need to build against the Pigeon Post API, in one place. Start with an SDK in your language, reach for the `pigeon` CLI for scripting and local development, and keep the glossary handy when the lore gets thick.

| Page | What it covers |
| --- | --- |
| [SDKs](/reference/sdks) | Official Node and Python libraries, community Go and Ruby clients, install commands, and minimal dispatch examples. |
| [CLI](/reference/cli) | The `pigeon` command-line tool — login, dispatch, tailing flights, sending flocks, and forwarding webhooks locally. |
| [Glossary](/reference/glossary) | Definitions for Aviary, Flight, Flock, Loft, Wingtime, and the rest of the flock vocabulary. |

Looking for endpoint-level detail? See the [API Reference](/api-reference) for the full, interactive OpenAPI documentation. Just getting started? Try the [Quickstart](/getting-started/quickstart).
